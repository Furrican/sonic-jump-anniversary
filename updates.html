Updates

Upload Date: 10/04/2025

Sonic Jump Anniversary is now released!

Sonic Jump Anniversary is a mod of the Android port of the original Sonic Jump to celebrate the 20th anniversary of Sonic Jump.
This mod makes this version of Sonic Jump playable on modern Android devices. But it also features a soundtrack select, new game modes, a multi-language feautre, and more!

GET THE LATEST VERSION

After downloading the update, close the game application before installing the update.
